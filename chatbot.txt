Fakultas Ilmu Komputer Universitas Indonesia atau disingkat Fasilkom UI adalah salah satu fakultas Rumpun Ilmu Sains dan Teknologi yang mengelola program pendidikan jenjang dengan gelar sarjana (S1), magister (S2), dan doktor (S3) serta program pelatihan non-gelar.

Berdasarkan sejarah, Fakultas Ilmu Komputer UI (Fasilkom) berdiri sejak tahun 1993, dan saat ini membuka 5 program studi untuk sarjana, 2 untuk program magister, dan 1 untuk program doktor, Sebelum pendiriannya, pendidikan Ilmu Komputer di UI dilaksanakan oleh Pusilkom UI (Pusat Ilmu Komputer) yang kini berstatus sebagai unit usaha akademis di bawah Fasilkom UI.

Kegiatan akademiknya adalah Pendidikan, Penelitian, dan Pengabdian kepada Masyarakat. 

Jika kau bertanya berapa program sarjana, ada 4 Program Sarjana di Fasilkom UI, yakni
1. Reguler dan Paralel Program Studi Ilmu Komputer
2. Reguler dan Paralel Program Studi Sistem Informasi
3. Kelas Internasional Program Studi Ilmu Komputer
4. Paralel untuk D3 (Ekstensi) Sistem Informasi.

Program Studi Ilmu Komputer di bawah naungan Fakultas Ilmu Komputer Universitas Indonesia (Fasilkom UI) memberikan kesempatan kepada para lulusan SMA/sederajat yang terbaik untuk mendapatkan pendidikan dan keahlian di bidang Ilmu Komputer pada tingkat sarjana (S1), Program Ilmu Komputer ini dirancang untuk memenuhi kebutuhan terhadap tenaga-tenaga muda yang terampil dan profesional di bidang Teknologi Informasi. 

Program Studi Sistem Informasi di bawah naungan Fakultas Ilmu Komputer Universitas Indonesia (Fasilkom UI) memberikan kesempatan kepada lulusan SMA/sederajat yang terbaik untuk mendapatkan pendidikan dan keahlian di bidang Sistem Informasi/Teknologi Informasi pada tingkat sarjana (S1), Program ini dirancang untuk memenuhi kebutuhan terhadap tenaga-tenaga muda yang terampil dan profesional, terutama terkait dengan pengembangan, pemanfaatan, dan pengelolaan Sistem Informasi/Teknologi Informasi dalam suatu organisasi. 

Untuk Program Kelas Internasional, Sejak tahun 2002, Fasilkom-UI bekerjasama dengan School of Information Technology and Electrical Engineering, The University of Queensland (ITEE-UQ) dalam menyelenggarakan Program Sarjana Internasional (S1-Kelas Internasional) bidang Ilmu Komputer dan Teknologi Informasi, Peserta program ini harus menyelesaikan delapan semester pendidikan yang dibagi menjadi dua bagian, yakni pada semester 1-5 diselenggarakan di UI dan pada semester 6-8 diselenggarakan di UQ, Australia, Seluruh proses pendidikan diselenggarakan dalam Bahasa Inggris. Kemudian, lulusan program ini akan mendapatkan dua gelar,yaitu Sarjana Ilmu Komputer dari UI, dan Bachelor of Information Technology dari ITEE UQ. 

Program Studi Sistem Informasi kelas Ekstensi D3 dilatarbelakangi oleh keinginan Fasilkom UI untuk menyediakan program peningkatan skill dan kompetensi bagi para lulusan program D3 di bidang-bidang yang berkaitan dengan teknologi informasi dan sudah bekerja, Kompetensi utama lulusan program ini adalah kemampuan menangani penerapan sistem dan teknologi informasi dalam suatu organisasi maupun mengembangkan penelitian, khususnya penelitian terapan teknologi informasi, di sektor bisnis dan sektor publik.

1. Magister(S2)
2. Doktor Ilmu Komputer(S3).

Program Magister terbagi 2, Magister Ilmu Komputer (MIK) dan Magister Teknologi Informasi (MTI).

Magister Ilmu Komputer adalah program dirancang sebagai research-based master program untuk mencetak tenaga ahli dengan wawasan riset dalam bidang ilmu komputer, Kegiatan perkuliahan di Magister Ilmu Komputer dikaitkan dengan kegiatan penelitian yang sedang berlangsung di laboratorium-laboratorium riset Fasilkom UI, yakni Lab Computational Intelligence, Lab Digital Library and Distance Learning, Lab Formal Metnods in Software Engineering, Lab High Performance Computing & Grid Computing, Lab Image Processing and Geographics Information Systems, Lab Information Retrieval, Lab IT Governance, Lab E-Government, dan Lab Enterprise Computing.

Magister Teknologi Informasi (MTI) adalah pendidikan tingkat lanjut (strata 2) mengenai penerapan Teknologi Informasi, Program Magister Teknologi Informasi dipimpin oleh seorang koordinator, yang melapor kepada Dekan Fakultas Ilmu Komputer Universitas Indonesia, Program Magister Teknologi Informasi dimulai tahun 1996 sebagai program kekhusuan Teknologi Informasi (TI) dan Sejak 2005, Program Magister Teknologi Informasi berdiri sebagai program sendiri dan lulusan berhak menyandang gelar Magister Teknologi Informasi.

Program Doktor Ilmu Komputer (DIK) merupakan program doktor yang pertama di Indonesia dalam bidang Ilmu Komputer. Program Doktor Ilmu Komputer ini dirancang untuk menghasilkan pakar Ilmu Komputer yang mampu menerapkan ilmunya, baik untuk penyelesaian masalah aplikatif maupun upengembangan ilmu komputer lebih lanjut. 

Fakultas Ilmu Komputer tentu memiliki fasilitas layaknya fasilitas di fakultas lain sebagai berikut:
    -Perpustakaan
    -Kantin dan Restoran Yoshinoya
    -Laboratorium
    -Ruang diskusi
    -Situs Web Perkuliahan
    -Aula dengan kapasitas maksimal 120 orang
    -Hotspot UI 24 jam.

Fakultas Ilmu Komputer mempunyai 9 laboratorium riset yang mengkhususkan untuk melakukan penelitian-penelitian di bidang-bidang sebagai berikut:
    -Computational Intelligence
    -Digital Libraries and Distance Learning
    -Formal Methods in Software Engineering
    -High Performance and Grid Computing
    -Image Processing and Geographical Information Systems
    -Text Processing and Information Retrieval
    -Enterprise Computing
    -IT Governance.
    -E-Government.
